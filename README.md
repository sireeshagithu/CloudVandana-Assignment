2. String Manipulation - Anagram Check 
Write a Java program that takes two strings as input and checks if they are anagrams 
(containing the same characters in a different order). 
Example: 
 
Input: "listen", "silent" 
Output: true 
Input: "hello", "world" 
Output: false
